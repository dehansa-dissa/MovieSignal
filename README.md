# 🎬 MovieSignal

**MovieSignal** is a responsive React web application where users can browse trending movies and view detailed information about each movie.

---

## 📌 About

This website allows users to:

- 📺 See a list of trending movies
- 🔍 Browse details like posters, ratings, and titles
- 📱 Enjoy a fully responsive design for desktop and mobile

---

## 🛠️ Built With

- **React** (Frontend)
- **CSS**
- **JavaScript**
- Public movie API
